# Statements
The **actions** that a program takes, are expressed as **statements**

Common actions include:

  **Declaring** a variable
  ```csharp
  int counter;
  ```

  **Assigning** a value
  ```csharp
  counter = 1;
  ```

  Declaring + **initializing**
  ```csharp
  int counter = 1;
  ```

  **Printing** a value
  ```csharp
  Console.WriteLine(counter);
  ```

  **Modifying** a value
  ```csharp
  counter++;
  ```
  
  ```csharp
  sum = a + b;
  ```
