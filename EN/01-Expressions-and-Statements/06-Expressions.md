# Expressions
**Expressions** == sequences of operators, literals and variables which are evaluated to a value

  * Consist of at least one **operand**
  * Can have 1 or more **operators**
```csharp
int y = x + 5;
```
```csharp
string name = "John Doe";
```
```csharp
int r = (150-20) / 2 + 5;
```
