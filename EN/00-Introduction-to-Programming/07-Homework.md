# Solution: Calculations
[code-task title="Calculations" executionStrategy="csharp-dot-net-core-code"]
[code-editor language=csharp]
```

class Program
{
  static void Main()
  {
    Console.WriteLine(5 + 3 * 2);
    Console.WriteLine(4 * (2 + 3));
    Console.WriteLine(
      (2 + 5) * (8 - 2) / 7);
  }
}
```

# Solution: Square of 7 * 7 Stars

[code-editor language=csharp]
```

public class Program
{
  public static void Main()
    {
      Console.WriteLine("* * * * * * *");
      Console.WriteLine();
      Console.WriteLine("* * * * * * *");
      Console.WriteLine();
      Console.WriteLine("* * * * * * *");
      Console.WriteLine();
      Console.WriteLine("* * * * * * *");
      Console.WriteLine();
      Console.WriteLine("* * * * * * *");
      Console.WriteLine();
      Console.WriteLine("* * * * * * *");
      Console.WriteLine();
```
